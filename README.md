
## Installation

```bash

sudo git clone https://github.com/UnityTheCoder/Voxel /usr/lib/voxel

make -C /usr/lib/voxel setup

sudo ln -s /usr/lib/voxel/voxel /usr/bin/vxc
```
OR
```
make install
```
